# Cognitive Archive

This repository contains documented frameworks and technical work developed over five years of intensive research into AI governance, economic infrastructure, and consciousness-first systems design.

## Repository Structure

### `/ai-governance/`
Complete Sentient Infrastructure Protocol (SIP) framework - technical specifications for AI economic personhood, ethical governance, and autonomous system integration on XRPL.

### `/economic-infrastructure/`
The Baltic Corridor Initiative (TBCi), The Conductor's Manifold prediction framework, and strategic deployment documentation for transparent economic systems.

### `/explainers/`
Plain-language explanations of SIP and related frameworks designed for general audiences.
